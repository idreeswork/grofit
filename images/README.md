## How to add images
1. Save with exact filename shown above
2. Drop into this /images/ folder
3. Commit and push to GitHub → Cloudflare auto-redeploys in ~60 seconds
4. If no image is present, a styled placeholder shows automatically

## Template screenshots — how to export from Canva
1. Open your template folder in Canva
2. Click on the template you want to show
3. File → Download → PNG, 1x
4. Rename to the filename above and drop it here
